Text Inputs

한 줄로 된 텍스트 인풋입니다. kp-input-item으로 사용합니다. state-required 또는 state-error클래스를 추가해 폼의 상태를 나타낼 수 있습니다. input 태그에 disabled나 readonly같은 속성을 지정해 줄 수도 있습니다.

Selectbox

옵션 목록이 나열되는 셀렉트 박스입니다. 너무 긴 레이블은 줄임 처리되므로, 한 줄에 표시되도록 간략하게 작성하는 것이 좋습니다. kp-selectbox로 사용합니다. select 태그에 disabled 속성을 지정해 줄 수도 있습니다.
